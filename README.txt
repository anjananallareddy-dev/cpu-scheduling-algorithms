CPU Scheduler Mini Project (C, Linked List, RR/FCFS/Priority)

Files:
 main.c - menu
 process.c/.h - process creation, deletion, queue mgmt
 scheduler.c/.h - scheduling logic

Build:
 gcc main.c process.c scheduler.c -o scheduler

Run:
 ./scheduler

Algorithms:
✅ FCFS
✅ Priority
✅ Round Robin

Use in VS Code:
 Open folder → Terminal → gcc compile command
